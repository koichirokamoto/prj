# gae-node
GAE/Node.js simple server with TypeScript

## Prerequisite

[intall yarn here](https://yarnpkg.com/lang/en/docs/install/#mac-stable)

[install gcloud sdk here](https://cloud.google.com/sdk/docs/downloads-versioned-archives)

## Install

yarn install

## Start in local

yarn start:local

## Deploy to GAE

yarn deploy
